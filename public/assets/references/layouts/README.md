# 상세페이지 레이아웃 레퍼런스

전환형 강의 상세페이지의 **레이아웃·카피·섹션 구성** 기준을 잡기 위한 벤치마크 캡처를 모은다.
(마스코트/컬러 같은 브랜드 아이덴티티 레퍼런스는 상위 폴더 `references/`·`robot/`에 있음)

## 넣는 법
- 파일명 예: `ref-01-클래스101.png`, `ref-02-인프런.png` … 순번 + 출처를 적는다.
- 가능하면 **풀페이지 세로 캡처**(상단 Hero부터 하단 Final CTA까지)로.
- 5장 내외. 모이면 이 표를 채워서 공통 패턴(섹션 순서·가격블록·신뢰요소·CTA 반복)을 정리하고
  그 합으로 **마스터 레이아웃**을 확정한다.
